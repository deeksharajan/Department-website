# Department-website (Student & Admin Department Portal)

A multi-page web platform built during my 1st semester to simulate distinct user privileges for educational management.
It is created using HTML, CSS and JavaScript.

## Live Demo
[https://github.com/deeksharajan/Department-website.git]

## Core Features
* Frontend role-based routing (Admin vs. Student dashboards)
* Dynamic data handling using JavaScript LocalStorage
* Form validation for credential inputs
